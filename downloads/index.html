ملف التثبيت المباشر

استخدم هذا الملف للتثبيت اليدوي على الهاتف. النسخة مبنية من نفس شفرة الويب المعتمدة في المستودع.

  • الإصدار: v9.2 RC
  • الحد الأدنى للنظام: Android 7.0+
  • نوع البناء: release (R8 + shrinkResources + v2 signature)
  • جديد هذه النسخة: إصلاح نهائي لصوت جلسة المحادثة الصوتية عبر Web Audio API يعمل فعليًا على Android WebView، إصلاح إذن الميكروفون في APK عبر دمج الإذن الأصلي مع إعادة محاولة آمنة وonPermissionRequest للـ WebView، زر الرجوع الهرمي بسلوك "اضغط مرتين للخروج" بدل الإزعاج بتأكيد دائم، تصحيح رصيد كود الترقية بحيث $5 تبقى $5 فعلاً في /me/quota، وتبسيط شريط الكتابة بإزالة "سطر" و"فقرة" واستبدالهما بزر المحادثة الصوتية وإعادة التوليد. لم نكسر البوابة أو OpenRouter أو دخول الإدارة أو الحصص لكل مستخدم.
